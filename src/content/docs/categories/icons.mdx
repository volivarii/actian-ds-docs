The DS-Kit ships **234 icons** organized into **11 semantic groups** derived from the Figma Actual-icons layout. Eight icons (`add`, `book-bookmark`, `dataset`, `directory`, `download`, `export`, `minimize`, `process`) belong to multiple groups and appear in each section below.

Group order is specific-first: more context-specific groups (Connector, Status, Cursor) appear before universal ones (Common). This is the same ordering used by the knowledge-side priority resolver in [`components/src/icon-groups.json`](https://github.com/volivarii/actian-ds-knowledge/blob/main/components/src/icon-groups.json).
