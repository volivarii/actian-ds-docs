## AI-native by default

Every page is also available as raw markdown for AI agents:

- <a href={`${base}llms.txt`}>`llms.txt`</a> — agent-readable site index
- <a href={`${base}llms-full.txt`}>`llms-full.txt`</a> — full content dump (6080 lines)
- `<page>.md` — every page has a `.md` twin (e.g. `/foundations/color.md`)
- `<meta name="actian-ds-*">` — slug, source, version, schema on every page

Designed for Claude Code, Cursor, and skill-based agents to consume the system without scraping HTML.
